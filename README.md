# Project_power_bi
Комплексный дашборд для анализа продаж, логистики и эффективности персонала.

**Ключевые показатели (на базе статуса Delivered):**
**Выручка:** 35,91 млн.
**Прибыль:** 29,45 млн при марже 82,01%.

**Эффективность:** Средняя выручка на сотрудника — 359,06 тыс.

**Инсайт:** Выявлена аномальная стоимость доставки в ВКО (12,7 тыс.).

**Меры:** Реализованы сложные DAX-вычисления (YOY%, чистые продажи, динамика клиентов).
